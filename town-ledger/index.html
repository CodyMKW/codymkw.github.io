What this editor covers

This edits the core Animal Crossing (GCN) save structure that Deluxe is built on top of: player names, wallet (bells, Nook's loan, post office savings), town name, birthday, gender/face, held item, shirt, inventory background, and your 15 pocket slots. Item fields are searchable by name against a community item list of ~2,367 entries — type to search, or type a hex ID directly. Anything shown as "Unknown Item" just isn't in that list; it's still safe to leave alone or set by hex. Deluxe-exclusive systems (town ordinances, e+ villagers, custom PWPs, skin tone) live in save regions that aren't publicly documented, so this editor leaves those bytes untouched — your Deluxe features will carry over safely, just not editable here yet. The checksum is recalculated automatically whenever you download, and both save copies in the file are kept in sync, exactly like the game does on its own.
