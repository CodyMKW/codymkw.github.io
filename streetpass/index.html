Frequently Asked Questions

What is this?
It's a retro handheld notification simulator built out of absolute boredom lol. It renders an accurate, high-definition reproduction of old-school 3D handheld notifications to use for profiles or memes.
Why isn't my avatar image showing up?
The image link provided must support CORS (Cross-Origin Resource Sharing) headers so the Canvas script can safely manipulate it. Try hosting the asset directly on your own web host dashboard or a public image repository.
