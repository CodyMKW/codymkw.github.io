Emulator Tips

  • Important: In-game save features (like in-game menus) will not persist between sessions. Use the Save State option in the EmulatorJS menu — this is saved automatically when you quit.
  • To export a raw .sav or .srm file, first trigger the game's native save, then use Export Save in the EmulatorJS menu to get the latest data.
  • Games with have a save state that loads automatically.
  • Games launched from your Library automatically save and restore state on quit.
  • Use the Export Saves button above to back up all your progress as a JSON file.
  • The toolbar auto-hides during play. Click Pin to keep it visible, or hover the top of the screen.
