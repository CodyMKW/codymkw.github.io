Cody @codymkw Admin 📢 Announcement Jul 1, 2026, 09:18 PM
📦 Codyverse Plaza is Closing

After some time running this community, I've made the decision to shut down Codyverse Plaza.

The main reason is practical — the KV database has been filling up faster than expected, and keeping it running long-term isn't something I want to maintain right now.

What happens next
  • The site will remain up for a short period so you can save anything you want to keep
  • A full archive of all posts, channels, and members will be preserved locally
  • Rather than activating shutdown mode, the entire site will be replaced with a permanent read-only archive — all posts, channels, replies, and members stay intact exactly as they are
Thank you

This was a fun little project and I appreciate everyone who signed up and posted. Whether you dropped in once or stuck around — it meant something.

— Cody (@codymkw)

Cody @codymkw Admin 📢 Announcement Jun 30, 2026, 07:12 PM
🛠️ Codyverse Plaza — Patch Notes v2 (June 30, 2026)
📢 Announcements Channel
  • New dedicated #announcements channel pinned to the top of the channel list
  • Admin-only posting — everyone else sees a lock notice in place of the composer
  • 6,000 character limit instead of 280 — patch notes and long-form updates fit comfortably
  • Full Markdown rendering: ## headers, bold, italic, strikethrough, inline code, fenced ` code blocks, - bullet lists, 1. ordered lists, > blockquotes, --- dividers, and [text](url) links
  • Markdown toolbar appears automatically when posting in #announcements — H2, H3, bold, italic, list, code, code block, quote buttons, plus a live preview toggle so you can see it rendered before posting
  • Regular users can still react, reply, and bookmark announcement posts
  • Posts in the channel get a teal highlight and a 📢 Announcement badge
🔗 Post Link Deep Linking (fixed)
  • Copying a post link now actually works end-to-end
  • On page load, the app reads the #channel/postId hash from the URL, switches to the correct channel, and scrolls to + highlights the linked post
  • Works for direct links shared externally and for in-app navigation (no full reload needed)
  • If the post no longer exists, a toast explains it instead of failing silently
🚦 Site Mode (Admin)
  • New Site Mode section in the Admin Panel with three states:

| Mode | Who sees it | What it shows |

|---|---|---|

| ✅ Normal | Everyone | Site runs normally |

| 🔧 Maintenance | All non-admins | Amber fullscreen overlay, optional message, pulsing "Back Soon" ticker |

| ☠️ Shutdown | All non-admins | Dark permanent closure screen, scanline texture, "Permanently Closed" ticker |

  • Optional custom message field for both maintenance and shutdown modes
  • Shutdown mode requires a confirmation dialog before activating
  • Admin always bypasses — you can still access and fix everything while the site is locked
  • Worker enforces the block at the API level — all non-admin requests return 503 when a mode is active
  • Open sessions are re-checked every 60 seconds — users already on the site get the overlay pushed to them automatically without refreshing
  • One-click Set Normal to lift any active mode instantly
Backend
  • New endpoints: /api/site-mode (GET, public), /api/admin/site-mode (POST, admin only)
  • Post creation now respects per-channel character limits (6000 for announcements, 280 everywhere else)
  • Post edit endpoint also respects the channel-aware limit
  • Announcements channel is protected from deletion alongside the other system channels

🛠️ Codyverse Plaza — Patch Notes (June 30, 2026)
Channels
  • Added channel icons — pick from 12 emoji when creating a new channel
  • Added channel favorites — star a channel to pin it to its own "Pinned" section at the top of the sidebar
  • Fixed the delete (✕) button rendering outside/below the channel row instead of beside it (was invalid nested <button> markup)
  • Channel list is now scrollable with a max height instead of pushing the page down as more get added
  • Added a filter/search box for channels (auto-appears once you have 6+)
  • Channels are now grouped into Default and Community sections, each with empty-state messaging
Posts
  • Edit posts — edit your own posts in place, shows an (edited) tag with hover timestamp
  • Quote posts — pulls a snippet into the composer as a styled quote block
  • Copy link to post
  • Report posts — flag another user's post with an optional reason, goes straight to the admin queue
  • Random post shuffle button (🎲)
  • Compact/comfortable feed density toggle (persists across sessions)
Profile & Progression
  • Level/XP system computed from lifetime post count — shown as a badge + progress bar on your own card and anyone's profile
  • Daily login streak tracker with milestone toasts (3/7/14/30/60/100 days)
  • Confetti burst on your first post of the day
Composer
  • Emoji quick-bar — 20 common emoji, one click to insert
  • Mention autocomplete (@username)
Quality of Life
  • Global mute toggle for sound effects
  • Back-to-top button
  • Image lightbox (click any post image to view full-size)
  • Bookmarks system with a dedicated modal
  • Trending posts sidebar
  • Keyboard shortcuts: N new post, / search, B bookmarks, Esc close modal
Admin Panel
  • Reported Posts section — view open reports with preview, reporter, reason, and one-click Dismiss or Delete Post
  • Live unread-report badge on the Admin button (polls every 45s)
  • User management now scans KV directly instead of relying on the registry, so all existing users show up
  • Channel management — delete custom channels (system channels protected)
  • Global broadcast notifications, direct messages to users, ban/unban, force logout all, wipe channel posts, telemetry export
Backend
  • New endpoints: /api/posts/edit, /api/posts/report, /api/admin/reports, /api/admin/reports/resolve, /api/admin/reports/delete-post, /api/admin/communities, /api/admin/communities/delete
  • Lifetime postCount now tracked per-user and returned from /api/me and /api/user/lookup
  • Channel objects now carry an icon field end-to-end
